#### type (mode)

| value        | description                          |
| ------------ | ------------------------------------ |
| `auto`       | auto clean                           |
| `border`     | border clean [^1]                    |
| `spot`       | spot clean [^1]                      |
| `spotArea`   | spot area and custom area clean [^2] |
| `customArea` | custom area clean [^2]               |
| `stop`       | stop                                 |

[^1]: Models without mapping functionality only
[^2]: Models with mapping functionality only

#### act (action)

| value    | description |
| -------- | ----------- |
| `start`  | start       |
| `pause`  | pause       |
| `resume` | resume      |
| `stop`   | stop        |
