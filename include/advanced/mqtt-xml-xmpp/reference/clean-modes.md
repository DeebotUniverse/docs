#### type (mode)

| value        | description                     |
| ------------ | ------------------------------- |
| `auto`       | auto clean                      |
| `border`     | border clean                    |
| `spot`       | spot clean                      |
| `SpotArea`   | spot area and custom area clean |
| `singleroom` | single room clean               |
| `stop`       | stop                            |

#### act (action)

| value | description |
| ----- | ----------- |
| `s`   | start       |
| `p`   | pause       |
| `r`   | resume      |
| `h`   | stop        |
